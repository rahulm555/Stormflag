1)This package includes the source code for the paper "STORMflag- A tool for alert and forecast of severe weather systems based on image processing and doppler weather radar raw data processing".
2)The pdf file named installation explains the step by step procedure to download, install the dependancies and test the source code.
3)Test data is placed inside the folder which has source code.Test data files are namely "max1.gif"&"rawfile.raw"
4)After the run of the program, result will be generated inside the folder "Stormflag" containing source code.Results will be inside a folder which bears the date of run of the script.
